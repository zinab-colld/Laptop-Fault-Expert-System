# Laptop Fault Expert System

This project is an Expert System to diagnose laptop issues using Python.
It includes:
- Expert Rules
- Inference Engine
- Solutions & Advice
- CLI interface

## How to run:
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py
